# AIO Test Automation Pilot

This project creates AIO Tests test cases using the AIO REST API.

## Current status

Working proof of concept:

- Project: TVSYSTEMS
- Created test case: TVSYSTEMS-TC-43
- Script type: Classic
- Tool: Python
- Runs from VS Code

## Files

- `create_test.py` - Python script that creates a test case in AIO
- `.env` - stores the AIO API token locally
- `venv/` - local Python virtual environment
